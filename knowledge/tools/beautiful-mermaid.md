# Beautiful Mermaid

Open source diagram rendering library built for the AI era. Ultra-fast, fully themeable, outputs to SVG and ASCII. Supports Flowchart, State, Sequence, Class, and ER diagrams.

## Key Features

- Ultra-fast rendering optimized for AI agent workflows
- Fully themeable with CSS custom properties
- Multiple output formats: SVG and ASCII
- Supports multiple diagram types: Flowchart, State, Sequence, Class, ER
- Open source and available at https://agents.craft.do/mermaid

## Use Cases

- AI agent reasoning and code comprehension visualization
- Diagram rendering in AI-augmented development workflows
- Creating publication-ready diagrams from programmatic input
- Themeable diagrams that adapt to user preferences

## Why It Matters

Balint Orosz built this because diagrams are becoming the primary way of reasoning about code with AI agents, but existing solutions weren't aesthetically satisfying for daily use. Beautiful Mermaid provides beautiful, fast diagram rendering specifically optimized for the AI era.
