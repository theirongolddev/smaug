Claude Code skill that integrates with multiple financial platforms to provide portfolio analysis and strategic financial decision support. Named after Charlie Munger's principle that capital discipline is a competitive advantage.

## Key Features

- Integration with Mercury, Stripe, ChartMogul, Ramp, and Google Sheets
- Portfolio and financial strategy analysis
- Expense monitoring and decision support
- Open-source and extensible
- Works with Claude Code and Codex

## Use Cases

- Analyzing company growth and expenses
- Making financial decisions (event planning, capital allocation)
- Controlling expenses through Ramp integration
- Understanding financial sources across platforms
- In-depth financial analysis for bootstrapped companies
