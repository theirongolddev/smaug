## Summary

Pencil is an infinite design canvas built with WebGL that integrates directly with Claude Code to turn designs into code. It enables design iteration in parallel with code generation through agentic design systems, with design files stored as JSON-based .pen format in git repositories. The tool eliminates the traditional design-to-code handoff friction by allowing designs to live alongside code and be edited by agents.

## Key Takeaways

- **Integrated design-code workflow**: Design canvas runs directly in your IDE (Claude Code or Cursor) rather than requiring separate design tools
- **Agentic design system**: Multiple design agents can iterate in parallel with the codebase, updating designs dynamically
- **Git-native format**: Design files are JSON-based .pen format that live in your repo, enabling version control and collaboration
- **WebGL performance**: Superfast canvas rendering enables real-time design iteration without latency
- **One-click implementation**: Turn designs into production code instantly with Claude Code's code generation
- **Local-first architecture**: No cloud dependency; designs stay in your git repo with full ownership
