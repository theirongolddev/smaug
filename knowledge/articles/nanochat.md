nanochat is the simplest experimental harness for training and deploying Large Language Models on a single GPU node under $1000 budget. The system provides a complete end-to-end pipeline from raw text to deployed chat interface, with breakthrough cost efficiency (e.g., training GPT-2-capable model for ~$72 vs $43K in 2019).

## Key Takeaways

- **Single Complexity Dial**: The entire system configured through one parameter (`--depth`, number of transformer layers), with all other hyperparameters automatically calculated using empirically-derived scaling laws
- **Miniseries Principle**: Architectural improvements must work across all model depths, ensuring principled and generalizable optimization
- **Time-to-GPT-2 Benchmark**: Uses wall-clock training time to reach GPT-2-grade capability (CORE score ~0.256+) as primary performance metric
- **Production-Ready Pipeline**: Covers tokenization, pretraining, finetuning, evaluation, inference with both CLI and ChatGPT-like web UI
- **Accessible Research**: Enables cost-effective LLM development ($20-72 on spot instances) and community contributions to speedrun leaderboard

<p>nanochat is a minimal experimental harness for training and deploying Large Language Models (LLMs). It is designed to be simple, hackable, and accessible, enabling end-to-end LLM development on a single GPU node with budgets under $1000. The system covers the complete pipeline from raw text to a deployed chat interface.</p>
<p><strong>What nanochat does:</strong></p>
<ul>
<li>Train compute-optimal models from scratch using a single complexity parameter</li>
<li>Fine-tune base models for conversational capabilities</li>
<li>Evaluate models using standardized benchmarks</li>
<li>Deploy models with production-ready serving infrastructure</li>
<li>Maintain reproducibility through systematic experimentation</li>
</ul>

<p>The entire system is configured through one primary parameter: <code class="rounded-sm bg-[#e5e5e5] px-[0.25rem] py-[0.20rem] text-xs font-normal leading-[15px] before:hidden after:hidden dark:bg-[#484848]/30">--depth</code>, which specifies the number of transformer layers.
